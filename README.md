# Техники тест-дизайна

Техники тест-дизайна — это системные методы, которые помогают тестировщикам создавать эффективные тестовые сценарии и тестовые данные для проверки работы программного продукта. 

Здесь я применил популярные техники тест-дизайна для тестирования https://qa.demoshopping.ru для эффективности тестовых сценариев для регистрации и авторизации, фильтрации и сортировки продуктов.
